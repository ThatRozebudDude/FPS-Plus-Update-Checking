# What Is This?

This is the repository that [FPS Plus](https://github.com/ThatRozebudDude/FPS-Plus-Public) checks to tell if it's up-to-date or not. Originally it checked a file in the FPS Plus repository but I decided to move it to an external repository so that the master branch of FPS Plus doesn't have to keep a reference to a version that is out-of-date with the code in that branch. The only thing here is the `latest` file, which is the version number for the latest release build of the game.

### This is currently not used by the current release version of FPS Pllus but will be in the next update.
